How It Works

1

Load Layer

Open QGIS and load your vector/shapefile.

2

Open Plugin

Find the icon in the toolbar or Plugins menu.

3

Configure (New!)

Choose your classification method (Jenks, Quantile), adjust colors, and label settings.

4

Visualize

Click OK to generate the map instantly or export the style.

Installation

01 Download the ZIP file from the button above.
02 Open QGIS → Plugins → Manage and Install.
03 Select "Install from ZIP" tab.
04 Browse file & Click "Install Plugin".
05 Enjoy Advanced Thematic Mapping!
